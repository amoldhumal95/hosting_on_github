Introduction

In this section you will learn how to create a static website using HTML. If you are new to HTML please refer W3School for basic understanding of HTML.

Quick Links:

  • HTML
  • CSS
  • JS

Prerequists

There is no requirement of prior knowledge.But you should be familiar with basics of HTML and related terms.

I will provide the reference code at the end of the section

Software

For the coding purpose any text editor is suitable. But to run website on localhost we require IDE.
Please prefer Visual Studio Code.
After the installation of Visual Studio Code Please follow the instructions as below:

  1. Open Visual Studio Code

  2. Click on Extensions Tab

  3. Search for "Live Server".

  4. Install the extension. ( Use ALT + L O key combination to launch the server )

Setup coding workspace

  1. Open File Explorer and create folder for your code.

  2. Include folders for css and js files.

  3. Open the same folder in visual studio code from file explorer.

    Or

    Press CTRL + K O key combination and select your folder.

  4. Create a new file with extension .html

Coding

Now let's start designing the webpage.
Below is the simple HTML code. 


<!DOCTYPE html>
<html lang="en">
<head>
<title>Homepage</title>
</head>
<body>

<h1>Welcome</h1>

</body>
</html>

To see the webpage on localhost press ALT + L O. It will open your default web browser.

Getting Started with GitHub

  • If you do not have Github account please create an account using below link.
    Sign up to Github.

  • If you have an existing account please login and follow the below steps
    Or
    refer the GitHub Guide

  1. First create a new repository for your site. Click on to create new repo.



  2. Provide a shore and simple name for the repository. Description is is optional but you can add it. Make sure the visibility of the repository is public.

    Hit the create repository button.




  3. On next page click on uploadig an existing file link. Next page will open to upload files to the repository. Simply open file explorer and drag all related files to the webpage. Within a moment all files will get uploaded and then click on Commit changes.





Enable Github-Pages

  • Next step is to enable the Github Pages.This will allow you to host your site live on web.
  • To enable github pages click on the setting button and scroll down to the Github Pages section.



    Click on the link shown in the Github Pages section. It will navigate you to the Github pages settings.

    Click on the dropdown list and select main branch and then click on save button.



    All done your site is now live and the url for the same is provided in the Github-Pages section.
